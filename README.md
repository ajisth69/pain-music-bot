# PAIN !! Music Bot 🎵

A next-generation, ultra-fast, lag-free Telegram Music Bot built with Pyrogram v2 and PyTgCalls.

### Features
- ⚡️ Lightning Fast Streaming (Zero Lag)
- 🎨 God-Level Aesthetic UI & Custom Fonts
- 🎵 JioSaavn Integration
- 🎛 Full VC Controls (Play, Pause, Skip, Stop)

### Quick Deploy
Deploy directly to Render using the button below:

[![Deploy to Render](https://render.com/images/deploy-to-render-button.svg)](https://render.com/deploy?repo=https://github.com/ajisth69/pain-music-bot)

### Manual Setup
1. Clone the repo: `git clone https://github.com/ajisth69/pain-music-bot`
2. Install requirements: `pip install -r requirements.txt`
3. Copy `.env.example` to `.env` and fill your credentials.
4. Run: `python main.py`
